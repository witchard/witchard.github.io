Here I will talk a little about the projects I am running on github.

galleryGen

Simple static web gallery generator that creates gallery files and thumbnails from folders on disk.

ServeDown

Generate wavedrom diagrams from your jupyter notebooks.

arDMX

This project provides the circuit diagram, arduino source and a simple python served html/javascript gui to enable the control of DMX lighting from a PC for minimal cost. Also supports uDMX.

magicChords

A simple javascript webapp to transpose chord charts. Will generate PDFs and print in a nice format too! Working version here.
